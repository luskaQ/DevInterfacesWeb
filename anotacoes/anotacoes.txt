BoxModel

Border -> borda (muitas vezes invisivel) que engloba o conteudo do elemento
Padding -> Espaço do conteudo ate sua borda 
Margin ->

           Borda
|            |
|<--margin-->|<--padding--> CONTEUDO
|            |


Ordem: Top, right, bottom, left

            ^
  Width     |
<-------->  | Height
            |

% é relativa ao tamanho do elemento pai 


box-sizing: quem vai ser afetado para manter o padding (borda aumentada ou elemento encolhido)

display: como o elemento se comporta (bloco, inline)
